# Multiplayer Fusion Unity 3D Project

https://roibru.itch.io/Week9

## Overview
This is a simple multiplayer game built with Unity 3D and Photon Fusion. 
It demonstrates basic multiplayer features like spawning, power-ups, and combat mechanics.

Move with arrows
Shoot with space
# What we added:
* boot multiplies speed by 2
* shield makes you invincible
* player dies when under 0 health
